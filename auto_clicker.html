About

This tool is used to generate one mouse click for every used specified interval.

Download

Click the link to download the file.

Features of the app:

  • Use keyboard short cuts to start/stop no matter which window you are in.
  • Customise the interval to your needs.
  • Displays how long the autoclicker has been running.
  • Colored status that displays if program is running or stopped.

Features to be added in the future:

  • Automatically stop the program after user desired time limit.
  • Add a dropdown to select which keyboard key/ mouse button to be simlated.
  • Add a help button.
