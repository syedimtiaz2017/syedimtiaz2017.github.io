Steps to follow (Refer figure 2):

  1. Click on “Add File” > Browse and select an excel file. Repeat the process to add as many excel files as you need.
  2. Once Excel files are added, the file names are displayed in the view box along with their file path.
  3. From the drop down, select the excel file.
  4. From the view box, click on the sheet that you want to compare.
  5. Follow step 3.
  6. Follow step 4.
  7. Click on “Browse…” and select the directory where you want your output to be.
  8. The output path is displayed.
  9. “Compare Sheets” allows you to compare the differences in table format. See Figure 3.
  10. “Compare Excels” allows you to compare the sheets and highlight the differences in excel 1 and save it as a new excel file. See Figure 4.
  11. “Merge Excels” allows you to merge two excel files (all sheets from excel 1 and excel 2 into one common file).
  12. Click “HELP” to view this help document.
